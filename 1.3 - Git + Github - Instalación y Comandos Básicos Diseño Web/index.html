¿Que es Git?

Es un sistema de versiones Es un entorno de trabajo

Diferentes ramas de versiones, se crea una carpeta 'oculta' en donde decidamos. O sea el proyecto deja de ser un archivo HTML y pasa a estar en una especie de lugar en donde voy a poder tener diferentes versiones de lo mismo.

Configurando Git

Podemos utilizartanto la terminal de nuestro sistema operativo como Git Bash (recomendado)

Iniciando sesión en Git

No lo mismo que hacerlo en Github, sino que se hace en Git, en mi computadora local.

Haciendo esto le doy al Git de mi computadora un nombre y un e-mail.

git config --global user.name "Tu nombre" git config --global user.email "Tu mail"

Comprobando la sesión

git config --get user.name git config --get user.email

Repositorio Locales

Es un almacenamiento local (por ahora) y virtual del proyecto.

Nos permite guardar versiones del código a los que podemos acceder cuando lo necesitemos.
